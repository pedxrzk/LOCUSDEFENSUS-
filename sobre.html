Sobre o Projeto

📡 Sistema IoT - Locus Defensus

O Locus Defensus é um sistema de monitoramento remoto desenvolvido para proporcionar segurança e controle através da tecnologia IoT (Internet das Coisas). Utilizando um microcontrolador ESP32 e diversos sensores, o sistema coleta dados ambientais em tempo real e os disponibiliza em uma interface web intuitiva e responsiva.

⚙️ Tecnologias Utilizadas

🔧

ESP32

Microcontrolador principal

🌡️

Sensores

Temperatura, Umidade, Pressão

🎨

Frontend

HTML5, CSS3, JavaScript

🔥

Auth, Database, Hosting

📱

Responsivo

⚡

IoT

MQTT / HTTP Protocol

🔐 Objetivos do Projeto

✅ Monitoramento contínuo de dados ambientais
✅ Controle remoto de dispositivos (LED, Buzzer)
✅ Interface intuitiva e de fácil navegação
✅ Sistema de autenticação seguro com Firebase
✅ Histórico de dados para análise
✅ Acessível de qualquer dispositivo com internet

📊 Funcionalidades

• Dashboard com visualização em tempo real de temperatura, umidade e pressão
• Controle manual de atuadores (LED e Buzzer)
• Histórico completo de dados com data e horário
• Autenticação segura com email/senha ou Google
• Interface responsiva para desktop e dispositivos móveis
• Dados persistidos em nuvem com Firebase
