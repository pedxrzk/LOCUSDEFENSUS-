Sobre o Projeto

Locus Defensus — Monitoramento IoT

Solução inteligente baseada em ESP32 que coleta temperatura, umidade e pressão atmosférica, transmitindo dados em tempo real para o Firebase Realtime Database.

ESP32 DevKit
BMP280
V1.0.0

Especificações Técnicas

  • Microcontrolador: ESP32-WROOM-32
  • Transmissão: Wi-Fi 2.4GHz
  • Armazenamento: Firebase NoSQL
  • Dashboard: HTML5 / CSS3 / JS

Objetivos do Sistema

  • Fornecer monitoramento remoto preciso e confiável em tempo real.
  • Garantir visualização rápida dos dados em qualquer dispositivo mobile.
  • Servir como base para futuras expansões de automação hospitalar.

Segurança e Backend

O sistema utiliza autenticação segura via Firebase Auth, garantindo que apenas usuários autorizados tenham acesso aos controles do Buzzer e LED. O backend processa as requisições com baixa latência, ideal para ambientes que exigem resposta imediata.
